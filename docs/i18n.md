# 多语言支持 (i18n)

Core UI 支持通过语言包文件实现界面多语言切换。

## 工作原理

1. `.ui` 文件中用 `@key` 引用翻译文字（而不是硬编码）
2. 语言包文件定义每个 key 的翻译
3. 运行时加载语言包，调用 `ApplyLanguage()` 更新全部控件

## .ui 文件中使用

在 `text` 属性前加 `@` 前缀引用翻译 key：

```xml
<Label text="@welcome_msg" />
<Button text="@btn_save" type="primary" />
<NavItem text="@nav_home" svg="..." />
<CheckBox text="@enable_feature" checked="true" />
<Toggle text="@dark_mode" />
<RadioButton text="@option_small" group="size" />
```

支持 `@key` 的控件：Label、Button、CheckBox、RadioButton、Toggle、NavItem。

不以 `@` 开头的文字照常显示，不受语言切换影响。

## 语言包文件格式

纯文本 `key=value` 格式，UTF-8 编码：

```ini
# lang/zh-CN.lang
# 注释以 # 或 ; 开头

welcome_msg=欢迎使用 Core UI
btn_save=保存
btn_cancel=取消
nav_home=首页
nav_settings=设置
enable_feature=启用功能
dark_mode=深色模式
option_small=小
```

```ini
# lang/en.lang
welcome_msg=Welcome to Core UI
btn_save=Save
btn_cancel=Cancel
nav_home=Home
nav_settings=Settings
enable_feature=Enable feature
dark_mode=Dark mode
option_small=Small
```

规则：
- 每行一个 `key=value`
- `#` 或 `;` 开头的行为注释
- 空行忽略
- key 不含空格，value 支持任意 UTF-8 字符
- 自动跳过 UTF-8 BOM

## C++ API

### 加载语言包

```cpp
// 从文件加载
g_layout.LoadLanguage(L"lang/zh-CN.lang");

// 从字符串加载
g_layout.LoadLanguageString("btn_save=Save\nbtn_cancel=Cancel");
```

### 应用语言

```cpp
// 遍历控件树，将所有 @key 替换为翻译后的文字
g_layout.ApplyLanguage();
```

### 查询单条翻译

```cpp
std::wstring text = g_layout.Tr("btn_save");  // 返回 L"保存"
// key 不存在时返回 key 本身
```

### 切换语言

```cpp
// 加载新语言包 → 应用 → 刷新界面
g_layout.LoadLanguage(L"lang/en.lang");
g_layout.ApplyLanguage();
ui_window_invalidate(g_win);
```

## 完整示例

**app.ui**
```xml
<ui version="1" width="800" height="600" title="My App">
  <VBox gap="0" expand="true">
    <TitleBar title="My App" />
    <VBox padding="24" gap="12" expand="true">
      <Label text="@welcome_msg" fontSize="20" bold="true" />
      <Button text="@btn_save" type="primary" onClick="onSave" />
      <HBox gap="8" height="32">
        <Label text="@settings_language" width="80" />
        <ComboBox items="中文,English" onChanged="onLangSwitch" />
      </HBox>
    </VBox>
  </VBox>
</ui>
```

**main.cpp**
```cpp
ui_init();

// 注册语言切换事件
g_layout.SetHandler("onLangSwitch", std::function<void(int)>([](int index) {
    const wchar_t* files[] = {L"lang/zh-CN.lang", L"lang/en.lang"};
    g_layout.LoadLanguage(files[index]);
    g_layout.ApplyLanguage();
    ui_window_invalidate(g_win);
}));

// 加载 UI
g_layout.LoadFile(L"app.ui");

// 加载默认语言
g_layout.LoadLanguage(L"lang/zh-CN.lang");
g_layout.ApplyLanguage();

// ... 创建窗口并显示
```

## 注意事项

1. **`@key` 仅支持 `text` 属性**：`placeholder`、`tooltip` 等属性目前不支持 `@key`
2. **语言包加载顺序**：`LoadLanguage()` 会清空之前加载的翻译，只保留最后一次加载的内容
3. **ApplyLanguage() 时机**：必须在 `LoadFile()` 之后调用（需要控件树已构建）
4. **key 未找到**：`Tr()` 和 `ApplyLanguage()` 遇到未翻译的 key 时，显示 key 本身
5. **热重载兼容**：`Reload()` 后需要重新调用 `ApplyLanguage()`
